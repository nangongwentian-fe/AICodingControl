## 各个AI Coding工具的Skills功能文档地址

1. claude code：https://code.claude.com/docs/en/skills
2. codex：https://developers.openai.com/codex/skills/
3. open code：https://opencode.ai/docs/skills/
4. antigravity：https://antigravity.google/docs/skills
5. trae/trae cn（trae cn和trae的区别其实就是文件夹名字不一样，trae：~/.trae, trae cn: ~/.trae-cn）：https://docs.trae.ai/ide/skills
6. cursor：https://cursor.com/cn/docs/context/skills

## 各个AI Coding工具的Skills目录

- Codex 全局目录：~/.codex/skills（文档写法为 $CODEX_HOME/skills，默认 macOS/Linux 即 ~/.codex/skills）。(developers.openai.com)
- Claude Code 全局目录：~/.claude/skills。(docs.claude.com)
- OpenCode 全局目录：~/.config/opencode/skill (opencode.ai)
- Cursor 用户自定义 Skills 目录：~/.cursor/skills；官方内置 Skills 目录：~/.cursor/skills-cursor（官方管理，不建议改动）。(effective-cursor.vercel.app)
- Antigravity 全局目录：~/.gemini/antigravity/skills。(antigravity.codes)
- Trae / Trae CN：~/.trae/skills 与 ~/.trae-cn/skills。
